# Micro Utilities (Next.js + Tailwind)

Fast everyday tools with no signup. Initial tools:
- Merge PDF
- Split PDF (range)
- Images → PDF
- QR Code Generator
- (Stubs) Compress PDF, MP4 → GIF

## Dev
```bash
npm i
npm run dev
```
